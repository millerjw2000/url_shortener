# url_shortener
A simple full stack application that allows users to create, manage, and share shortened links. Contains user registration using JWT authentication. Users can create shortened URLs which they can use to redirect them to the original associated site.

# Tech used
- Front end
  - React
  - React-router-dom
  - axios
  - CSS
- Back end
  - Spring
  - JWT authentication
  - PostgreSQL
